# Question: How can we use Python to analyze genomic data for gene expression profiling?

## Solution

In this tutorial, we will use Python with the `pandas`, `numpy`, and `matplotlib` libraries to analyze gene expression data. We will perform data loading, preprocessing, normalization, and visualization of gene expression levels.

### Prerequisites
Make sure you have the following libraries installed:

```bash
pip install pandas numpy matplotlib seaborn
```

### Step 1: Load the Gene Expression Data

Let's assume you have a CSV file named `gene_expression.csv` with the following structure:

| Gene    | Sample1 | Sample2 | Sample3 |
|---------|---------|---------|---------|
| GeneA  | 150     | 200     | 170     |
| GeneB  | 90      | 85      | 100     |
| GeneC  | 300     | 320     | 310     |

Here’s how to load the data:

```python
import pandas as pd

# Load the gene expression data
data = pd.read_csv('gene_expression.csv')
print(data.head())
```

### Step 2: Normalize the Data

Normalization helps in reducing technical variance. We will use Z-score normalization here.

```python
from scipy.stats import zscore

# Normalize the data
data_normalized = data.copy()
data_normalized.iloc[:, 1:] = data_normalized.iloc[:, 1:].apply(zscore, axis=1)
print(data_normalized.head())
```

### Step 3: Identify Differentially Expressed Genes

Assuming we have two samples to compare, we can calculate the fold change and identify differentially expressed genes.

```python
import numpy as np

# Calculate fold change between Sample1 and Sample2
data['FoldChange'] = (data['Sample2'] + 1) / (data['Sample1'] + 1)  # Adding 1 to avoid division by zero

# Define a threshold for differential expression
threshold = 1.5
data['DEG'] = np.where(abs(data['FoldChange']) > threshold, 'Yes', 'No')

print(data[['Gene', 'FoldChange', 'DEG']])
```

### Step 4: Visualize the Results

We will create a bar plot to visualize the expression levels of differentially expressed genes.

```python
import matplotlib.pyplot as plt
import seaborn as sns

# Filter to only DEGs for visualization
degs = data[data['DEG'] == 'Yes']

# Bar plot of differential expression
plt.figure(figsize=(10, 6))
sns.barplot(x='Gene', y='Sample1', data=degs)
plt.title('Differentially Expressed Genes between Sample1 and Sample2')
plt.xlabel('Gene')
plt.ylabel('Expression Level in Sample1')
plt.xticks(rotation=45)
plt.show()
```

### Conclusion

In this tutorial, we successfully loaded genomic data, normalized it, identified differentially expressed genes, and visualized the results using Python. This forms a basic framework you can expand upon for deeper research in bioinformatics.
