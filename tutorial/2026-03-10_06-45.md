# How can we use Python for Genome Wide Association Studies (GWAS) Data Analysis?

## Solution

In this tutorial, we'll perform a basic analysis of GWAS data using Python. We'll examine how to read GWAS summary statistics, filter significant SNPs, and visualize the results using Manhattan and Q-Q plots.

### Step 2: Import Libraries

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy import stats
```

### Step 3: Load GWAS Summary Statistics

Assuming you have a CSV file containing GWAS summary statistics with columns `SNP`, `P`, and `CHR` (chromosome):

```python
gwas_data = pd.read_csv('gwas_summary_statistics.csv')
gwas_data.head()
```

### Step 4: Filter Significant SNPs

Set a threshold for significance (commonly p < 5e-8):

```python
significant_snps = gwas_data[gwas_data['P'] < 5e-8]
print(significant_snps)
```

### Step 5: Create a Manhattan Plot

```python
def manhattan_plot(data):
    data['-log10(P)'] = -np.log10(data['P'])
    
    # Assign colors based on chromosome
    data['color'] = np.where(data['CHR'] % 2 == 0, 'blue', 'red')

    plt.figure(figsize=(12, 6))
    sns.scatterplot(data=data, x='SNP', y='-log10(P)', hue='color', legend=None)
    
    plt.title('Manhattan Plot')
    plt.xlabel('SNPs')
    plt.ylabel('-log10(P-value)')
    plt.xticks(rotation=90)
    plt.axhline(y=-np.log10(5e-8), color='grey', linestyle='--')
    plt.show()

manhattan_plot(gwas_data)
```

### Step 6: Create a Q-Q Plot

```python
def qq_plot(data):
    observed = -np.log10(np.sort(data['P']))
    expected = -np.log10(np.linspace(1/len(data), 1, len(data)))
    
    plt.figure(figsize=(6, 6))
    plt.scatter(expected, observed, color='blue')
    plt.plot([0, max(expected)], [0, max(expected)], color='red', linestyle='--')
    
    plt.title('Q-Q Plot')
    plt.xlabel('Expected -log10(P)')
    plt.ylabel('Observed -log10(P)')
    plt.xlim(0, max(expected))
    plt.ylim(0, max(observed))
    plt.show()

qq_plot(gwas_data)
```
