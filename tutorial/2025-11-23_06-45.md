# Question
How can we use Python to perform sequence alignment of DNA sequences and visualize the results using a Heatmap?

# Solution

### Step 1: Install Required Libraries
Make sure you have the following libraries installed: `biopython`, `numpy`, `matplotlib`, and `seaborn`. You can install them using pip:

```bash
pip install biopython numpy matplotlib seaborn
```

### Step 2: Import Required Modules

```python
from Bio import pairwise2
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
```

### Step 3: Define DNA Sequences

```python
# Sample DNA sequences
seq1 = "AGCTGAC"
seq2 = "AGCTGCC"
```

### Step 4: Perform Pairwise Sequence Alignment

```python
# Perform the alignment
alignments = pairwise2.align.globalxx(seq1, seq2)
```

### Step 5: Extract Alignment Scores

```python
# Extract scores from alignments
scores = []
for alignment in alignments:
    score = alignment[2]  # Score is the third element
    scores.append(score)
```

### Step 6: Create a Matrix for Heatmap

```python
# Create a distance matrix based on scores
distance_matrix = np.array(scores).reshape(len(scores), 1)
```

### Step 7: Visualize the Results with a Heatmap

```python
# Create a heatmap from the distance matrix
plt.figure(figsize=(6, 4))
sns.heatmap(distance_matrix, annot=True, cmap="YlGnBu", cbar_kws={'label': 'Alignment Score'})
plt.title("DNA Sequence Alignment Scores Heatmap")
plt.xlabel("Alignment")
plt.ylabel("Sequences")
plt.xticks(ticks=np.arange(1, 1+len(scores)), labels=['Alignment 1'], rotation=45)
plt.yticks(ticks=np.arange(1, 1+len(scores)), labels=['Seq1 vs Seq2'], rotation=0)
plt.show()
```

### Conclusion
This tutorial illustrates the process of sequence alignment using Python and visualizes the alignment scores using a Heatmap, which is essential in bioinformatics for understanding genetic relationships.
