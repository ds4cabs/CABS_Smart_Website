# Question: How to Use Python for Genome Sequence Alignment using Biopython?

Genome sequence alignment is crucial in bioinformatics for comparing genetic variations and similarities among organisms. Here's a detailed guide on performing a basic pairwise sequence alignment using the Biopython library.

## Solution

### Prerequisites

Ensure you have Biopython installed. You can install it using pip:

```bash
pip install biopython
```

### Code

```python
from Bio import SeqIO
from Bio import pairwise2
from Bio.pairwise2 import format_alignment

# Step 1: Load the DNA sequences from FASTA files
def load_sequences(file_path):
    sequences = {}
    for record in SeqIO.parse(file_path, "fasta"):
        sequences[record.id] = str(record.seq)
    return sequences

# Step 2: Perform pairwise alignment
def align_sequences(seq1, seq2):
    alignments = pairwise2.align.globalxx(seq1, seq2)
    return alignments

# Step 3: Format and display alignments
def display_alignments(alignments):
    for alignment in alignments:
        print(format_alignment(*alignment))

# Main execution
if __name__ == "__main__":
    # Load sequences from a FASTA file
    sequences = load_sequences("sequences.fasta")

    # Assume we have two sequences to align
    seq1 = sequences["sequence1_id"]  # Replace with actual ID
    seq2 = sequences["sequence2_id"]  # Replace with actual ID

    # Align the two sequences
    alignments = align_sequences(seq1, seq2)

    # Display the alignments
    display_alignments(alignments)
```

### Explanation

1. **Loading Sequences**: The `load_sequences` function reads sequences from a specified FASTA file and stores them in a dictionary.
  
2. **Aligning Sequences**: The `align_sequences` function takes two sequences and uses Biopython's `pairwise2` module to perform a global alignment.
  
3. **Displaying Alignments**: The `display_alignments` function formats the alignments in a readable manner using the `format_alignment` method.

### Example Input File (`sequences.fasta`)

```
>sequence1_id
AGCTACGTAG
>sequence2_id
AGCTTGCTAG
```

### Output

The output will display the aligned sequences with scores indicating their similarity.

```plaintext

  | || |  |
  AGCT-GCTAG
  Score=8
```
