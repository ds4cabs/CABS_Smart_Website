import numpy as np

### Step 2: Prepare the Dataset

Load your protein dataset (FASTA format) and process it.

```python
def load_proteins(file_path):
    sequences = []
    for record in SeqIO.parse(file_path, "fasta"):
        sequences.append(str(record.seq))
    return sequences

sequences = load_proteins("proteins.fasta")
print(f"Loaded {len(sequences)} protein sequences.")
```

### Step 3: Feature Extraction

Convert sequences into numerical representations (one-hot encoding or embeddings).

```python
def one_hot_encode(sequence, amino_acids='ACDEFGHIKLMNPQRSTVWY'):
    encoding = np.zeros((len(sequence), len(amino_acids)))
    for i, amino_acid in enumerate(sequence):
        if amino_acid in amino_acids:
            encoding[i, amino_acids.index(amino_acid)] = 1
    return encoding

X = np.array([one_hot_encode(seq) for seq in sequences])
```
