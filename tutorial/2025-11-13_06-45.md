#### Step 3: Feature Extraction
Convert sequences to numerical features:
```python
def extract_features(sequences):
    feature_list = []
    
    for seq in sequences:
        feature = []
        for aa in seq:
            feature.append(ord(aa) - 65)  # Convert amino acids to numerical values
        feature_list.append(feature)
        
    return pd.DataFrame(feature_list)
```

#### Step 4: Prepare Dataset
```python
# Load data
sequences, labels = load_data('protein_sequences.fasta')

# Extract features
features = extract_features(sequences)

# Split dataset into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(features, labels, test_size=0.2, random_state=42)
```

#### Step 5: Train Machine Learning Model
```python
# Initialize Random Forest Classifier
rf_model = RandomForestClassifier(n_estimators=100, random_state=42)

# Train the model
rf_model.fit(X_train, y_train)
```

#### Step 6: Evaluate the Model
```python
# Make predictions
y_pred = rf_model.predict(X_test)

# Evaluate performance
accuracy = accuracy_score(y_test, y_pred)
report = classification_report(y_test, y_pred)

print(f"Accuracy: {accuracy}")
print(f"Classification Report:\n{report}")
```
