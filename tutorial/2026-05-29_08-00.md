## Question
How can we use Python and machine learning to predict protein secondary structure from amino acid sequences?

## Solution

We will use the `scikit-learn` and `Biopython` libraries to create a basic model that predicts the secondary structure of proteins based on their amino acid sequences.

### Step 1: Data Preparation

We will use the CB513 dataset, which contains protein sequences and their corresponding secondary structure. Let's load and preprocess this data.

```python
import pandas as pd
from Bio import SeqIO
import numpy as np

# Load dataset
def load_data(file_path):
    sequences = []
    structures = []
    
    for record in SeqIO.parse(file_path, "fasta"):
        sequences.append(str(record.seq))
        structures.append(record.description.split()[1])  # Assuming secondary structure is in the description

    return sequences, structures

# Example file
file_path = 'cb513.fasta'  # Replace with your dataset path
sequences, structures = load_data(file_path)
```

### Step 2: Feature Extraction

Convert sequences into numerical features using a simple encoding scheme.

```python
from sklearn.preprocessing import OneHotEncoder

def encode_sequences(sequences):
    # Create a one-hot encoder
    encoder = OneHotEncoder(sparse=False)
    
    # Encode the sequences
    encoded_sequences = encoder.fit_transform(np.array(list("".join(sequences))).reshape(-1, 1)).reshape(len(sequences), -1, 20)
    
    # Aggregate over sequence length
    return np.mean(encoded_sequences, axis=1)

encoded_features = encode_sequences(sequences)
```

### Step 3: Train-Test Split

Split the dataset into training and testing sets.

```python
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(encoded_features, structures, test_size=0.2, random_state=42)
```

### Step 4: Model Training

We'll use a Random Forest Classifier to predict the secondary structure.

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import classification_report

# Initialize the model
model = RandomForestClassifier(n_estimators=100, random_state=42)

# Fit the model
model.fit(X_train, y_train)

# Predictions
y_pred = model.predict(X_test)

# Classification report
print(classification_report(y_test, y_pred))
```

### Step 5: Interpretation of Results

Evaluate the model performance based on the classification report output.

```python
# You can visualize the results more effectively with confusion matrices or other plots.
from sklearn.metrics import confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

cm = confusion_matrix(y_test, y_pred, labels=model.classes_)
sns.heatmap(cm, annot=True, fmt='d', cmap='Blues', xticklabels=model.classes_, yticklabels=model.classes_)
plt.ylabel('Actual')
plt.xlabel('Predicted')
plt.title('Confusion Matrix')
plt.show()
```
