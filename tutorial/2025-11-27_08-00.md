# How Can We Use Python for Sequence Alignment in Bioinformatics?

Sequence alignment is a fundamental task in bioinformatics, used to identify similarities between DNA, RNA, or protein sequences. This tutorial will guide you through the implementation of the Needleman-Wunsch algorithm for global sequence alignment using Python.

## Solution

### Step 1: Install Required Libraries

Ensure you have `numpy` installed. If not, you can install it using pip:

```bash
pip install numpy
```

### Step 2: Implement the Needleman-Wunsch Algorithm

```python
import numpy as np

# Define a scoring system
MATCH = 2
MISMATCH = -1
GAP = -2

def create_score_matrix(seq1, seq2):
    rows = len(seq1) + 1
    cols = len(seq2) + 1
    score_matrix = np.zeros((rows, cols))

    # Initialize first row and column
    for i in range(1, rows):
        score_matrix[i][0] = score_matrix[i-1][0] + GAP
    for j in range(1, cols):
        score_matrix[0][j] = score_matrix[0][j-1] + GAP

    return score_matrix

def fill_score_matrix(seq1, seq2, score_matrix):
    for i in range(1, len(seq1) + 1):
        for j in range(1, len(seq2) + 1):
            match = score_matrix[i-1][j-1] + (MATCH if seq1[i-1] == seq2[j-1] else MISMATCH)
            delete = score_matrix[i-1][j] + GAP
            insert = score_matrix[i][j-1] + GAP
            score_matrix[i][j] = max(match, delete, insert)

def align_sequences(seq1, seq2):
    score_matrix = create_score_matrix(seq1, seq2)
    fill_score_matrix(seq1, seq2, score_matrix)
    
    return score_matrix

def print_alignment(score_matrix, seq1, seq2):
    # Traceback to find the optimal alignment
    alignment1 = ""
    alignment2 = ""
    i, j = len(seq1), len(seq2)

    while i > 0 and j > 0:
        if score_matrix[i][j] == score_matrix[i-1][j-1] + (MATCH if seq1[i-1] == seq2[j-1] else MISMATCH):
            alignment1 += seq1[i-1]
            alignment2 += seq2[j-1]
            i -= 1
            j -= 1
        elif score_matrix[i][j] == score_matrix[i-1][j] + GAP:
            alignment1 += seq1[i-1]
            alignment2 += "-"
            i -= 1
        else:
            alignment1 += "-"
            alignment2 += seq2[j-1]
            j -= 1
    
    # Complete the alignment
    while i > 0:
        alignment1 += seq1[i-1]
        alignment2 += "-"
        i -= 1
    while j > 0:
        alignment1 += "-"
        alignment2 += seq2[j-1]
        j -= 1

    return alignment1[::-1], alignment2[::-1]

# Driver code
if __name__ == "__main__":
    seq1 = "AGCT"
    seq2 = "AGT"
    score_matrix = align_sequences(seq1, seq2)
    alignment1, alignment2 = print_alignment(score_matrix, seq1, seq2)

    print("Alignment:")
    print(alignment1)
    print(alignment2)
```

### Step 3: Test the Code

Run the above code, and you will see the optimal alignment of the sequences.
