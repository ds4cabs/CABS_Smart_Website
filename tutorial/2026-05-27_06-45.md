# How Can We Use Python for Advanced AI in Bioinformatics?

## Question
How can we implement a machine learning model to predict protein functions from sequences using Python?

## Solution

In this tutorial, we will use the `Biopython` library for sequence handling and `scikit-learn` for machine learning to predict protein functions based on sequence data. We will follow these steps:

1. Load and preprocess data.
2. Encode protein sequences.
3. Train a machine learning model.
4. Evaluate the model's performance.

### Step 1: Load and Preprocess Data

First, install the required libraries if you haven't already:

```bash
pip install biopython scikit-learn pandas
```

Now, we can start coding!

```python
import pandas as pd
from Bio import SeqIO

# Load protein sequences from a FASTA file
def load_sequences(file_path):
    sequences = {}
    for record in SeqIO.parse(file_path, "fasta"):
        sequences[record.id] = str(record.seq)
    return sequences

# Example usage
sequences = load_sequences('protein_sequences.fasta')
```

### Step 2: Encode Protein Sequences

We will encode the protein sequences using one-hot encoding.

```python
import numpy as np

def one_hot_encode(sequences):
    amino_acids = 'ACDEFGHIKLMNPQRSTVWY'
    n_sequences = len(sequences)
    max_length = max(len(seq) for seq in sequences.values())
    
    X = np.zeros((n_sequences, max_length, len(amino_acids)), dtype=int)
    seq_labels = list(sequences.keys())
    
    for i, seq in enumerate(sequences.values()):
        for j, aa in enumerate(seq):
            if aa in amino_acids:
                X[i, j, amino_acids.index(aa)] = 1
    return X, seq_labels

# Example usage
X, seq_labels = one_hot_encode(sequences)
```

### Step 3: Train a Machine Learning Model

We'll use a simple neural network with Keras to train on encoded sequences.

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense, Dropout
from sklearn.model_selection import train_test_split

# Assume we have labels for the sequences
labels = np.random.randint(0, 2, size=(len(seq_labels),))  # Random binary labels for demonstration

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.2, random_state=42)

# Build the model
model = Sequential()
model.add(LSTM(128, input_shape=(X.shape[1], X.shape[2])))
model.add(Dropout(0.5))
model.add(Dense(1, activation='sigmoid'))

model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])

# Train the model
model.fit(X_train, y_train, epochs=10, batch_size=32, validation_split=0.1)
```

### Step 4: Evaluate Model Performance

We can evaluate the model on the test set.

```python
loss, accuracy = model.evaluate(X_test, y_test)
print(f'Test Accuracy: {accuracy:.2f}')
```
