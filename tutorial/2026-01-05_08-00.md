# Advanced AI in Bioinformatics: Gene Expression Classification

## Question
How can we use machine learning to classify gene expression data for cancer diagnosis?

## Solution

### Introduction
In this tutorial, we will use Python to build a machine learning model that classifies gene expression data to predict cancer types. We will utilize libraries such as `pandas`, `scikit-learn`, and `numpy`.

### Prerequisites
Ensure you have the following libraries installed:


### Step 1: Load the Gene Expression Dataset
We will load a dataset containing gene expression features and labels for various cancer types.

```python
import pandas as pd

# Load the dataset
data = pd.read_csv('gene_expression_data.csv')  # Replace with your dataset path
print(data.head())
```

### Step 2: Data Preprocessing
Separate features and labels, handle missing values, and encode categorical labels.

```python
# Separate features and labels
X = data.drop('cancer_type', axis=1)
y = data['cancer_type']

# Handle missing values
X.fillna(X.mean(), inplace=True)

# Encode categorical labels
from sklearn.preprocessing import LabelEncoder

label_encoder = LabelEncoder()
y_encoded = label_encoder.fit_transform(y)
```

### Step 3: Train-Test Split
We will split the dataset into training and testing sets.

```python
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y_encoded, test_size=0.2, random_state=42)
```

### Step 4: Model Selection
We will use a Random Forest Classifier for classification.

```python
from sklearn.ensemble import RandomForestClassifier

# Initialize the model
model = RandomForestClassifier(n_estimators=100, random_state=42)

# Train the model
model.fit(X_train, y_train)
```

### Step 5: Model Evaluation
Evaluate the model performance using accuracy and confusion matrix.

```python
from sklearn.metrics import accuracy_score, confusion_matrix

# Make predictions
y_pred = model.predict(X_test)

# Calculate accuracy
accuracy = accuracy_score(y_test, y_pred)
print(f'Accuracy: {accuracy * 100:.2f}%')

# Confusion Matrix
cm = confusion_matrix(y_test, y_pred)
print('Confusion Matrix:')
print(cm)
```

### Step 6: Feature Importance
Analyze the importance of different genes in classification.

```python
import matplotlib.pyplot as plt
import numpy as np

# Get feature importances
importances = model.feature_importances_
indices = np.argsort(importances)[::-1]

# Plot the feature importances
plt.figure()
plt.title("Feature importances")
plt.bar(range(X.shape[1]), importances[indices], align="center")
plt.xticks(range(X.shape[1]), X.columns[indices], rotation=90)
plt.xlim([-1, X.shape[1]])
plt.show()
```
