### Step 3: Load and Prepare the Protein Sequence

We will use Biopython to prepare our protein sequence for structure prediction.

```python
from Bio import SeqIO

# Load a protein sequence from a FASTA file
def load_sequence(fasta_file):
    with open(fasta_file) as file:
        for record in SeqIO.parse(file, "fasta"):
            return str(record.seq)

sequence = load_sequence("protein.fasta")
print("Loaded Sequence:", sequence)
```
