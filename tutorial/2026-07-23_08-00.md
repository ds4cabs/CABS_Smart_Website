# Question
How can we use Python for sequence alignment in bioinformatics, specifically implementing the Needleman-Wunsch algorithm for global alignment?

# Solution

## Introduction
The Needleman-Wunsch algorithm is a dynamic programming algorithm used for sequence alignment in bioinformatics. It is particularly useful for aligning protein or nucleotide sequences.

## Implementation Steps
1. Define the scoring matrix for matches, mismatches, and gaps.
2. Initialize the scoring and traceback matrices.
3. Fill in the scoring matrix based on the alignment scores.
4. Trace back through the scoring matrix to get the optimal alignment.

## Code

```python
import numpy as np

# Function to create scoring matrix
def create_score_matrix(seq_a, seq_b, match=1, mismatch=-1, gap=-2):
    n = len(seq_a) + 1
    m = len(seq_b) + 1
    
    # Initialize score and traceback matrices
    score = np.zeros((n, m))
    traceback = np.zeros((n, m), dtype='object')

    # Fill the first row and column of the score matrix
    for i in range(1, n):
        score[i][0] = score[i-1][0] + gap
        traceback[i][0] = 'up'  # traceback direction

    for j in range(1, m):
        score[0][j] = score[0][j-1] + gap
        traceback[0][j] = 'left'  # traceback direction

    # Fill the score matrix and the traceback matrix
    for i in range(1, n):
        for j in range(1, m):
            match_score = score[i-1][j-1] + (match if seq_a[i-1] == seq_b[j-1] else mismatch)
            delete_score = score[i-1][j] + gap
            insert_score = score[i][j-1] + gap
            
            score[i][j] = max(match_score, delete_score, insert_score)

            # Update the traceback matrix
            if score[i][j] == match_score:
                traceback[i][j] = 'diag'
            elif score[i][j] == delete_score:
                traceback[i][j] = 'up'
            else:
                traceback[i][j] = 'left'

    return score, traceback

# Function to perform traceback and create aligned sequences
def traceback_aligned_sequences(traceback, seq_a, seq_b):
    align_a = []
    align_b = []
    i, j = len(seq_a), len(seq_b)

    while i > 0 or j > 0:
        if traceback[i][j] == 'diag':
            align_a.append(seq_a[i-1])
            align_b.append(seq_b[j-1])
            i -= 1
            j -= 1
        elif traceback[i][j] == 'up':
            align_a.append(seq_a[i-1])
            align_b.append('-')
            i -= 1
        else:  # 'left'
            align_a.append('-')
            align_b.append(seq_b[j-1])
            j -= 1

    return ''.join(reversed(align_a)), ''.join(reversed(align_b))

# Function to align two sequences
def needleman_wunsch(seq_a, seq_b):
    score, traceback = create_score_matrix(seq_a, seq_b)
    aligned_seq_a, aligned_seq_b = traceback_aligned_sequences(traceback, seq_a, seq_b)
    return aligned_seq_a, aligned_seq_b, score[-1][-1]

# Example usage
sequence_a = "AGGTAB"
sequence_b = "GXTXAYB"

aligned_a, aligned_b, final_score = needleman_wunsch(sequence_a, sequence_b)

print("Aligned Sequences:")
print(aligned_a)
print(aligned_b)
print("Final Alignment Score:", final_score)
```

## Conclusion
The above implementation provides a complete approach to performing global sequence alignment using the Needleman-Wunsch algorithm. The code captures the scores for the alignment and provides the final aligned sequences alongside their score.
