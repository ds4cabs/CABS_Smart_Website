# Tutorial: Advanced AI in Bioinformatics - Predicting Protein Structure Using Deep Learning

## Question
How can we use deep learning to predict the secondary structure of proteins from their amino acid sequences in Python?

## Solution

### Step 1: Install Necessary Libraries
Ensure you have the following libraries installed. You can install them using pip:


### Step 2: Import Required Libraries
```python
import numpy as np
import pandas as pd
from Bio import SeqIO
from sklearn.preprocessing import LabelEncoder
from keras.models import Sequential
from keras.layers import Dense, LSTM, Embedding, Dropout
from keras.utils import to_categorical
```

### Step 3: Load and Preprocess the Data
Assuming you have a dataset of protein sequences and their corresponding secondary structures.
```python
def load_data(file_path):
    sequences = []
    structures = []
    for record in SeqIO.parse(file_path, "fasta"):
        sequences.append(str(record.seq))
        structures.append(record.description.split('|')[-1])  # assuming structure is in the description
    return sequences, structures

# Example usage
sequences, structures = load_data("protein_data.fasta")
```

### Step 4: Encode the Data
```python
# Encode amino acids and structures
amino_acids = set("".join(sequences))
encoder = LabelEncoder()
encoder.fit(list(amino_acids))
encoded_sequences = [encoder.transform(list(seq)) for seq in sequences]

encoded_structures = LabelEncoder()
encoded_structures.fit(np.unique(structures))
encoded_labels = [encoded_structures.transform(list(structure)) for structure in structures]
```

### Step 5: Prepare Input and Output for the Neural Network
```python
max_len = 100  # Maximum length of protein sequences
X = np.array([np.pad(seq, (0, max_len - len(seq)), 'constant') if len(seq) < max_len else seq[:max_len] for seq in encoded_sequences])
Y = np.array([to_categorical(label, num_classes=len(encoded_structures.classes_)) for label in encoded_labels])
```

### Step 6: Build the Model
```python
model = Sequential()
model.add(Embedding(input_dim=len(encoder.classes_), output_dim=64, input_length=max_len))
model.add(LSTM(128, return_sequences=True))
model.add(Dropout(0.5))
model.add(LSTM(64))
model.add(Dense(len(encoded_structures.classes_), activation='softmax'))

model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])
```

### Step 7: Train the Model
```python
model.fit(X, np.array(Y), epochs=50, batch_size=32, validation_split=0.2)
```

### Step 8: Evaluate the Model
```python
loss, accuracy = model.evaluate(X, np.array(Y))
print(f'Loss: {loss}, Accuracy: {accuracy}')
```

### Step 9: Predictions
```python
def predict_structure(sequence):
    encoded_sequence = encoder.transform(list(sequence))
    padded_sequence = np.pad(encoded_sequence, (0, max_len - len(encoded_sequence)), 'constant')[:max_len]
    prediction = model.predict(np.array([padded_sequence]))
    predicted_structure = np.argmax(prediction, axis=2)
    return encoded_structures.inverse_transform(predicted_structure.flatten())

# Example usage
predicted_structure = predict_structure("ACDEFGHIKLMNPQRSTVWY")  # Sample amino acid sequence
print(predicted_structure)
```
