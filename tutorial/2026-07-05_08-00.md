# Question
How can we use Python to analyze DNA sequences for patterns using the Biopython library?

## Solution

### Prerequisites
Make sure you have Python and Biopython installed. You can install Biopython using pip:

```bash
pip install biopython
```

### Step 1: Import Necessary Libraries

```python
from Bio import SeqIO
from Bio.Seq import Seq
from Bio.SeqUtils import GC
```

### Step 2: Load DNA Sequence Data

Assume you have a FASTA file named `example.fasta`. This is how you can load it:

```python
def load_sequences(file):
    sequences = []
    for record in SeqIO.parse(file, "fasta"):
        sequences.append(record.seq)
    return sequences

dna_sequences = load_sequences("example.fasta")
```

### Step 3: Calculate GC Content

You can analyze the GC content of each sequence, which is important for understanding the stability of the DNA:

```python
def calculate_gc_content(sequences):
    return {str(seq): GC(seq) for seq in sequences}

gc_contents = calculate_gc_content(dna_sequences)
print(gc_contents)
```

### Step 4: Find Patterns in the DNA Sequence

You may want to find specific motifs (like "ATG" or "TAA") within the DNA sequences:

```python
def find_motif(sequences, motif):
    results = {}
    for i, seq in enumerate(sequences):
        positions = []
        start = 0
        while True:
            start = seq.find(motif, start)
            if start == -1: 
                break
            positions.append(start)
            start += 1
        results[f"Sequence_{i+1}"] = positions
    return results

motif_results = find_motif(dna_sequences, "ATG")
print(motif_results)
```

### Step 5: Visualize GC Content

You can visualize GC content using `matplotlib`. Make sure to install it first:

```bash
pip install matplotlib
```

Then use the following code to plot the GC content:

```python
import matplotlib.pyplot as plt

def visualize_gc_content(gc_contents):
    labels = gc_contents.keys()
    values = gc_contents.values()

    plt.bar(labels, values)
    plt.ylabel('GC Content (%)')
    plt.title('GC Content of DNA Sequences')
    plt.xticks(rotation=45)
    plt.show()

visualize_gc_content(gc_contents)
```
