# Question: How can we use Python to analyze genomic sequences and identify potential mutations associated with a disease?

## Solution

In this tutorial, we will use Python with libraries such as Biopython and Pandas to analyze genomic sequences and identify potential mutations. We will work with a hypothetical dataset containing reference and sample genomic sequences.

### Step 1: Load Reference and Sample Sequences

We will start by loading a reference genomic sequence and a sample sequence to compare for potential mutations.

```python
from Bio import SeqIO

# Load reference sequence
reference_seq = SeqIO.read("reference_genome.fasta", "fasta")

# Load sample sequence
sample_seq = SeqIO.read("sample_genome.fasta", "fasta")

print(f"Reference Sequence: {reference_seq.seq}")
print(f"Sample Sequence: {sample_seq.seq}")
```

### Step 2: Identify Mutations

Next, we will create a function to compare the reference sequence with the sample sequence and identify the positions and types of mutations.

```python
def identify_mutations(reference, sample):
    mutations = []
    for index, (ref_base, sample_base) in enumerate(zip(reference, sample)):
        if ref_base != sample_base:
            mutations.append((index + 1, ref_base, sample_base))  # Position, Reference base, Sample base
    return mutations

# Find mutations
mutations = identify_mutations(reference_seq.seq, sample_seq.seq)
print("Identified Mutations:")
for mutation in mutations:
    position, ref_base, sample_base = mutation
    print(f"Position: {position}, Reference: {ref_base}, Sample: {sample_base}")
```

### Step 3: Analyze Mutation Types

Now that we have identified mutations, we will categorize them into types (e.g., substitutions, insertions, deletions).

```python
def categorize_mutations(mutations):
    categories = {'substitution': [], 'insertion': [], 'deletion': []}
    
    for index, ref_base, sample_base in mutations:
        if len(ref_base) == len(sample_base):
            categories['substitution'].append((index, ref_base, sample_base))
        elif len(sample_base) > len(ref_base):
            categories['insertion'].append((index, ref_base, sample_base))
        elif len(sample_base) < len(ref_base):
            categories['deletion'].append((index, ref_base, sample_base))

    return categories

# Categorize identified mutations
mutation_categories = categorize_mutations(mutations)

print("\nCategorized Mutations:")
for category, entries in mutation_categories.items():
    print(f"{category.capitalize()}:")
    for entry in entries:
        print(entry)
```

### Step 4: Visualization (Optional)

For better insights, you can visualize the mutations using a simple bar plot.

```python
import matplotlib.pyplot as plt

def visualize_mutations(mutation_categories):
    categories = list(mutation_categories.keys())
    counts = [len(mutation_categories[cat]) for cat in categories]

    plt.bar(categories, counts, color=['blue', 'orange', 'green'])
    plt.xlabel('Mutation Type')
    plt.ylabel('Count')
    plt.title('Mutation Types in Genomic Sequences')
    plt.show()

# Visualize mutation types
visualize_mutations(mutation_categories)
```
