### Question
How can we use Python to analyze DNA sequence data for identifying gene mutations using the Biopython library?

### Solution

#### Step 1: Install Biopython
Make sure to install the Biopython library if you haven't already.

```bash
pip install biopython
```

#### Step 2: Import required libraries

```python
from Bio import SeqIO
from Bio.Seq import Seq
from Bio.Align import MultipleSeqAlignment
import pandas as pd
```

#### Step 3: Load DNA sequences

Assume we have a FASTA file named `sequences.fasta` containing DNA sequences.

```python
def load_sequences(file_path):
    sequences = []
    for record in SeqIO.parse(file_path, "fasta"):
        sequences.append((record.id, str(record.seq)))
    return sequences

sequences = load_sequences("sequences.fasta")
```

#### Step 4: Analyze for mutations

We will compare a reference sequence against other sequences to identify mutations.

```python
def find_mutations(reference_seq, other_seqs):
    mutations = {}
    for seq_id, seq in other_seqs:
        mutation_positions = []
        for i in range(len(reference_seq)):
            if reference_seq[i] != seq[i]:
                mutation_positions.append(i)
        mutations[seq_id] = mutation_positions
    return mutations

# Example reference sequence
reference_sequence = sequences[0][1]  # Assuming the first one is the reference
mutations = find_mutations(reference_sequence, sequences[1:])
```

#### Step 5: Output mutations

Let's create a DataFrame to display the mutations found.

```python
def display_mutations(mutations):
    mutation_data = []
    for seq_id, positions in mutations.items():
        mutation_data.append({"Sequence ID": seq_id, "Mutation Positions": positions})
    
    df = pd.DataFrame(mutation_data)
    return df

mutation_df = display_mutations(mutations)
print(mutation_df)
```

#### Step 6: Visualize mutations (optional)

You can visualize the mutations using matplotlib.

```python
import matplotlib.pyplot as plt

def plot_mutations(mutation_df):
    seq_ids = mutation_df['Sequence ID']
    mutation_counts = mutation_df['Mutation Positions'].apply(len)

    plt.bar(seq_ids, mutation_counts)
    plt.title('Number of Mutations per Sequence')
    plt.xlabel('Sequence ID')
    plt.ylabel('Number of Mutations')
    plt.xticks(rotation=45)
    plt.show()

plot_mutations(mutation_df)
```
