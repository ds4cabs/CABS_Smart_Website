# How Can We Use Python for Advanced AI in Bioinformatics?

In this tutorial, we will explore how to utilize Python for advanced AI applications in bioinformatics. We will specifically focus on building a simple machine learning model to classify DNA sequences using the `scikit-learn` library.

## Problem Statement

Given a dataset of DNA sequences labeled as different classes (e.g., genes, non-coding regions), how can we build a machine learning model to classify these sequences based on their features?

## Solution

### Step 1: Setting up the environment

Make sure you have the required libraries installed:

```bash
pip install numpy pandas scikit-learn biopython
```

### Step 2: Import Necessary Libraries

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import LabelEncoder
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, classification_report
from Bio.Seq import Seq
```

### Step 3: Load the Dataset

Assuming you have a CSV file `dna_sequences.csv` with two columns: `sequence` (DNA sequences) and `label` (class labels).

```python
# Load the dataset
data = pd.read_csv('dna_sequences.csv')

# Display head of the dataset
print(data.head())
```

### Step 4: Feature Extraction

Convert DNA sequences into numerical features. A simple approach is to count nucleotide occurrences.

```python
def extract_features(sequence):
    seq = Seq(sequence)
    return [seq.count('A'), seq.count('T'), seq.count('C'), seq.count('G')]

# Apply feature extraction
features = data['sequence'].apply(extract_features)
features = np.array(features.tolist())
```

### Step 5: Encoding Labels

Convert the textual labels into numerical format.

```python
label_encoder = LabelEncoder()
labels = label_encoder.fit_transform(data['label'])
```

### Step 6: Split Data

Split the dataset into training and testing sets.

```python
X_train, X_test, y_train, y_test = train_test_split(features, labels, test_size=0.2, random_state=42)
```

### Step 7: Build the Model

Create and train a Random Forest Classifier.

```python
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
```

### Step 8: Make Predictions

Evaluate the model on the test data.

```python
predictions = model.predict(X_test)

# Evaluate the model
accuracy = accuracy_score(y_test, predictions)
report = classification_report(y_test, predictions, target_names=label_encoder.classes_)

print(f'Accuracy: {accuracy}')
print('Classification Report:')
print(report)
```
