# How can we use Python for sequence alignment in Bioinformatics?

## Solution

In this tutorial, we will use the `Bio` package from Biopython to perform sequence alignment. We'll demonstrate how to align two DNA sequences using the Needleman-Wunsch algorithm.

### Prerequisites

Make sure you have Biopython installed. You can install it using pip:

```bash
pip install biopython
```

### Import Libraries

```python
from Bio import pairwise2
from Bio.pairwise2 import format_alignment
```

### Define Sequences

Let's define two DNA sequences that we want to align.

```python
seq1 = "AGCTGAC"
seq2 = "AGCTAC"
```

### Perform Alignment

Now, we will perform the Needleman-Wunsch alignment.

```python
alignments = pairwise2.align.globalxx(seq1, seq2)
```

### Display Alignments

We will now display the alignments in a human-readable format.

```python
for alignment in alignments:
    print(format_alignment(*alignment))
```

### Full Code

Here is the complete code for the sequence alignment:

```python
from Bio import pairwise2
from Bio.pairwise2 import format_alignment

# Define sequences
seq1 = "AGCTGAC"
seq2 = "AGCTAC"

# Perform alignment
alignments = pairwise2.align.globalxx(seq1, seq2)

# Display alignments
for alignment in alignments:
    print(format_alignment(*alignment))
```

### Output

After running the code, you should see an alignment result similar to the following:

```
AGCTGAC
| ||||
AGCT-AC
  Score=5
```

This output shows the aligned sequences along with the alignment score, indicating the degree of similarity between them.
