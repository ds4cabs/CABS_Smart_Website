# Question: How can we use Python to analyze genomic sequences and identify potential mutations?

## Solution

In this tutorial, we will use Python to perform basic genomic sequence analysis by identifying mutations in a given DNA sequence. We will leverage the Biopython library for handling biological data.

### Prerequisites

Make sure you have Python installed along with Biopython. You can install Biopython using pip:

```bash
pip install biopython
```

### Step 1: Import Libraries

```python
from Bio import SeqIO
from Bio.Seq import Seq
```

### Step 2: Load the Reference and Mutated DNA Sequences

Assume you have a reference sequence and a mutated sequence stored in FASTA format. We will load these sequences into our Python script.

```python
# Load the reference sequence
with open("reference_sequence.fasta") as reference_file:
    reference_seq = SeqIO.read(reference_file, "fasta").seq

# Load the mutated sequence
with open("mutated_sequence.fasta") as mutated_file:
    mutated_seq = SeqIO.read(mutated_file, "fasta").seq
```

### Step 3: Identify Mutations

We will compare the reference sequence with the mutated sequence to identify positions where mutations occur.

```python
def identify_mutations(ref_seq, mut_seq):
    mutations = []
    
    # Ensure both sequences are of the same length
    if len(ref_seq) != len(mut_seq):
        raise ValueError("Reference and mutated sequences must be of the same length.")

    for i in range(len(ref_seq)):
        if ref_seq[i] != mut_seq[i]:
            mutations.append((i, ref_seq[i], mut_seq[i]))  # Store index and mutated bases
            
    return mutations

# Get mutations
mutations = identify_mutations(reference_seq, mutated_seq)
```

### Step 4: Display Results

Now we can print the mutations identified in the sequences.

```python
if mutations:
    print("Identified Mutations (Position, Ref Base, Mut Base):")
    for mutation in mutations:
        print(f"Position: {mutation[0]}, Ref Base: {mutation[1]}, Mut Base: {mutation[2]}")
else:
    print("No mutations identified.")
```

### Example Input Files

**reference_sequence.fasta**
```
>Reference
ACGTACGTACGTACGT
```

**mutated_sequence.fasta**
```

```

### Step 5: Run the Script

Once you've created the script, run it in your terminal or script runner. The output will show any mutations compared to the reference sequence.

### Conclusion

This Python script provides a straightforward method to identify mutations in genomic sequences using the Biopython library. As you become more comfortable, you can expand this analysis to include multiple sequences and various types of mutations.
